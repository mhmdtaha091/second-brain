# AR Interface Design

Designing for augmented reality means rethinking interaction paradigms:
screens become space, clicks become gestures.

## Design Principles

- **Spatial first** — interfaces occupy meaningful positions in 3D space
- **Gesture-driven** — pinch to select, grab to move, swipe to dismiss
- **Glanceable** — depth, color, size, and motion encode meaning faster than text

## Hand Tracking

MediaPipe provides 21 hand landmarks per hand, enabling:

- Pinch detection (thumb–index distance)
- Fist/grab detection (joint-angle finger-curl analysis)
- Pose estimation (palm normal, twist angle)

## Visual Language for Graphs in AR

- Node size = connectivity
- Node color = community
- Edge glow = activity
- Spatial clustering = semantic similarity
