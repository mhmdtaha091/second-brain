# Zettelkasten Method

A note-taking system built on atomic notes and dense linking, popularized by
sociologist Niklas Luhmann's card-index system.

## Principles

1. **Atomicity** — one idea per note
2. **Autonomy** — each note stands on its own
3. **Linking** — always connect new notes to existing ones
4. **Your own words** — rewriting is understanding

## Why Links Beat Folders

A note about "graph layouts" could live under *math*, *visualization*, or
*design*. Folders force one choice; links allow all three.
