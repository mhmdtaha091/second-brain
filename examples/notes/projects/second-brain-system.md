# Second Brain System

A local-first knowledge management system with 3D/AR visualization.

## Core Principles

- **100% local** — all data stays on your machine
- **Markdown native** — plain text, future-proof, any editor works
- **AR-ready** — explore your knowledge as a graph floating in space
- **Hand control** — navigate with gestures, no controllers needed

## Architecture

1. **Notes** — markdown files with YAML frontmatter and `[[wikilinks]]`
2. **Graph engine** — parses notes, builds the graph, detects communities
3. **Visualization** — Three.js force-directed 3D graph with a WebXR AR mode
4. **CLI** — quick capture, linking, and search from the terminal
